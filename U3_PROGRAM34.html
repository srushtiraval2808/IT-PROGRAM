HTML Text Formatting Examples

Bold text: This uses the <b> tag for visual thickness.

Strong text: This uses <strong> for semantic importance (SEO/Screen readers).

Italic text: This uses the <i> tag for visual tilt.

Emphasized text: This uses <em> to stress a word or phrase.

Marked text: Use <mark> to highlight text with a yellow background.

Smaller text: Use <small> for side comments or copyright info.

Deleted text: Use <del> to show a strike-through (often for old prices).

Inserted text: Use <ins> to underline text added to a document.
